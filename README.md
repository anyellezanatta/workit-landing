# workit-landing

This is an implementation of the workit landing challenge from frontendmentor using Vite Vanilla.

[WebApp](https://anyellezanatta.github.io/workit-landing)

## Stack

- Modern CSS
- Stylelint
- Prettier

## Running

```bash
yarn dev
```

Follow the instuctions on the terminal.

## Linting

```bash
yarn lint:css
```

## Formatting

```bash
yarn format
```
